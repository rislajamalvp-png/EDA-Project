# Indian Personal Finance & Spending — EDA Project

## Project Overview
A complete Exploratory Data Analysis on financial and demographic 
data of 20,000 individuals in India, focusing on income, expenses, 
savings behaviour and lifestyle patterns.

**Tools:** Python, Jupyter Notebook, Pandas, NumPy, Matplotlib, Seaborn, SciPy

---


## Key Insights
1. Income is right-skewed — most people earn modest incomes
2. Savings rate is normally distributed — people save 20-35% consistently
3. City Tier significantly affects savings (T-test p=0.0000)
4. Income level does NOT affect savings rate (ANOVA p=0.1316)
5. More dependents = lower savings rate (-0.22 correlation)
6. Tier 3 cities save 3x more than Tier 1 cities

---

## Dataset
- **Source:** Kaggle — Indian Personal Finance & Spending Habits
- **Size:** 20,000 rows × 27 columns
- **Type:** Numerical + Categorical
